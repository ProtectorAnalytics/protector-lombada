Como funciona

Do radar ao email em segundos

Processo 100% automatizado. Nenhuma intervencao manual necessaria apos a instalacao.

1
Camera detecta
A camera ALPHADIGI le a placa e mede a velocidade com radar integrado.
2
Dados enviados
Placa, velocidade e foto sao transmitidos automaticamente via HTTPS.
3
Sistema processa
O servidor identifica a camera, salva os dados e verifica o limite de velocidade.
4
Alerta enviado
Se excedeu o limite, gera PDF com foto e envia por email para os responsaveis.
Integracao

Instalacao simples, operacao autonoma

Uma URL para todas as cameras. O sistema identifica cada uma automaticamente pelo serial.

URL unica universal
Todas as cameras de todos os clientes apontam para o mesmo endereco. Sem tokens, sem configuracao extra.
Identificacao por serial
A camera envia o numero de serie automaticamente no payload. O sistema faz o match com o cadastro.
3 passos para ativar
Cadastrar cliente, cadastrar camera com serial, apontar a camera para a URL. Pronto.
